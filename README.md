# ICVE Helper

智慧职教MOOC学习助手，仅支持智慧职教MOOC平台。

## 功能特性

- 自动学习功能
- AI智能答题
- 支持多种AI模型
- 现代化UI设计

## 更新日志

### v1.3.0 (2025-11-28)
- **页面适配优化**
  - 根据URL自动显示对应功能：学习页面显示「学习+日志」，答题页面显示「答题+日志」
  - 默认标签页根据页面类型自动切换
- **日志系统增强**
  - 答题时显示完整题目信息（题号、题型、题目内容、选项）
  - 所有配置变更都会记录日志（播放倍速、等待时间、AI模型等）
  - 优化日志页面布局，清空按钮移至底部
  - 修复日志占位符不消失的问题
- **多选题修复**
  - 修复多选题只选中一个选项的问题（点击间隔太快）
  - 每个选项点击后等待200ms确保状态更新
- **学习功能优化**
  - 移除暂停学习功能，简化操作流程
  - 面板折叠状态持久化保存
- **代码清理**
  - 删除未使用的 `Logger.enable/disable` 方法
  - 删除未使用的 `DOMCache.getAll/clear` 方法
  - 删除未使用的 `ConfigManager.set/setAIConfig` 方法
  - 删除未使用的 `DOMCache.panel` 预缓存对象
  - 删除 `stopLearning` 函数及相关代码
  - 移除重复的配置处理逻辑
