# ICVE Helper

智慧职教MOOC学习助手，仅支持智慧职教MOOC平台。

## 功能特性

- 自动学习功能
- AI智能答题
- 支持多种AI模型
- 现代化UI设计

## 更新日志

  - 优化章节展开逻辑：先通过API获取内容，再点击箭头展开

  - 使用对象映射替代switch语句，提升事件处理性能

- **界面动画优化**
  - 调整按钮hoer效果，减小抬升幅度更细腻

- **性能提升**
  - 点击事件加入100ms防抖，防止误触发
