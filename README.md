# 智慧职教全能助手

一款功能强大的智慧职教平台辅助工具，集成自动学习和AI智能答题功能。

## 功能特性

### 自动学习模式

- **视频/音频自动播放** - 支持1x-16x倍速播放
- **文档自动翻页** - 可配置翻页间隔时间
- **章节自动展开** - 智能识别并展开下一章节
- **学习进度保存** - 自动记录已完成的学习节点
- **考试节点跳过** - 自动识别并跳过考试/测验类型节点
- **静音播放** - 可选静音模式，不打扰工作

### AI智能答题

- **多AI模型支持**
  - 通义千问 (Qwen)
  - OpenAI GPT
  - DeepSeek
  - Moonshot AI (Kimi)
  - 智谱AI (GLM)
  - 自定义API

- **多题型支持**
  - 单选题
  - 多选题
  - 判断题
  - 填空题

- **答题配置**
  - 可调节答题间隔
  - 自动交卷选项
  - 自定义API地址和模型

1. 安装浏览器扩展（任选其一）：

   - [Violentmonkey](https://violentmonkey.github.io/) - 开源轻量的选择
   - [ScriptCat（脚本猫）](https://scriptcat.org/) - 国产脚本管理器，功能丰富

## 使用说明

### 学习模式

1. 进入智慧职教课程学习页面
2. 面板自动显示在右上角
3. 点击 **"扫描"** 按钮扫描当前章节的学习节点
4. 点击 **"开始学习"** 启动自动学习
5. 可随时点击 **"暂停学习"** 停止

#### 学习配置说明

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 播放倍速 | 视频/音频播放速度 | 1.0x |
| 完成等待 | 节点完成后等待时间 | 8秒 |
| 翻页间隔 | 文档自动翻页间隔 | 15秒 |
| 展开延迟 | 章节展开等待时间 | 3秒 |
| 静音 | 是否静音播放媒体 | 关闭 |

### 答题模式

1. 进入考试/测验页面
2. 切换到 **"答题"** 标签页
3. 选择AI模型并配置API Key
4. 点击 **"开始答题"** 启动自动答题

#### AI配置说明

| AI模型 | API地址 | 推荐模型 |
|--------|---------|----------|
| 通义千问 | dashscope.aliyuncs.com | qwen-max |
| OpenAI | api.openai.com | gpt-4o-mini |
| DeepSeek | api.deepseek.com | deepseek-chat |
| Kimi | api.moonshot.cn | moonshot-v1-8k |
| 智谱AI | open.bigmodel.cn | glm-4 |

## 界面预览

面板采用现代玻璃拟态设计：

- 极光渐变主题配色
- 毛玻璃背景效果
- 流畅的动画过渡
- 支持浅色/深色主题切换
- 可拖动定位

## 配置存储

所有配置通过 `GM_setValue` / `GM_getValue` 持久化存储：

- 学习进度自动保存
- 已处理节点记录
- AI密钥安全存储
- 主题偏好保存

## 注意事项

1. **API Key 安全** - 请妥善保管您的API密钥，不要泄露给他人
2. **网络要求** - AI答题功能需要稳定的网络连接
3. **合理使用** - 建议配合自主学习使用，提高学习效率
4. **浏览器兼容** - 推荐使用 Chrome、Edge、Firefox 最新版本

## 快捷操作

- **拖动面板** - 按住标题栏拖动调整位置
- **折叠面板** - 点击右上角 `-` 按钮折叠/展开
- **切换主题** - 点击月亮/太阳图标切换深浅主题
- **重置进度** - 点击重置按钮清空学习记录
